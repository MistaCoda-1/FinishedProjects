Data Types

int
→ Whole numbers (e.g. 5, -12)

double
→ Decimal numbers (e.g. 3.14)

char
→ A single character (e.g. 'A', '7')

string
→ A string of characters (e.g. "Hello")

bool
→ true or false values

If / Else Statements

if (condition) {
        // runs if condition is true
} else if (otherCondition) {
        // runs if that’s true
} else {
        // runs if none are true
}

Comparison operators:
== equal   != not equal
< less than  > greater than
<= less or equal >= greater or equal

    // while loop

    // repeat while true
}

───────────────────────────────

Switch Statements

switch (value) {
  case option1:
        // code
    break;
  case option2:
        // code
    break;
  default:
        // fallback
    break;
}

→ Only works with int, char, or enum types

Sleep / Pause / Exit

#include <windows.h>
Sleep(2000);        // 2 second pause
system("pause");    // press any key to continue
exit(0);            // end program early
return 0;           // end main normally

Helpful Reminders

→ Use ; at the end of most lines
→ ' ' for char, " " for string
→ == compares, = assigns
→ cin >> and cout << are your best friends
→ Always wrap logic blocks in { }
